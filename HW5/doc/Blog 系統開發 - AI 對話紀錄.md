## 技術規格

### 功能

| 功能 | 說明 |
|------|------|
| 註冊 | 建立新帳號 |
| 登入 | 使用帳號密碼登入 |
| 登出 | 清除登入狀態 |
| 文章列表 | 顯示所有文章，含作者資訊 |
| 新增文章 | 需登入才能發布 |
| 編輯文章 | 需登入且為作者才能編輯 |
| 刪除文章 | 需登入且為作者才能刪除 |
| 文章詳情 | 點擊標題以彈窗顯示完整內容 |

### API

| 方法 | 路徑 | 說明 |
|------|------|------|
| POST | /api/register | 註冊新用戶 |
| POST | /api/login | 登入 |
| GET | /api/user | 檢查登入狀態 |
| GET | /api/posts | 取得所有文章 |
| GET | /api/posts/:id | 取得單篇文章 |
| POST | /api/posts | 新增文章（需登入） |
| PUT | /api/posts/:id | 更新文章（需登入） |
| DELETE | /api/posts/:id | 刪除文章（需登入） |

### 資料表

**users**
| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER | 主鍵 |
| username | TEXT | 帳號（唯一） |
| password | TEXT | 密碼（SHA256 hash） |
| created_at | DATETIME | 建立時間 |

**posts**
| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER | 主鍵 |
| user_id | INTEGER | 作者ID |
| title | TEXT | 標題 |
| content | TEXT | 內容 |
| created_at | DATETIME | 建立時間 |
